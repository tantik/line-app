#### Задача 1.3: Форма обратной связи (СРЕДНЯЯ)
**Описание:** Реализовать отправку лидов с landing page  
**Интеграция:** Отправка в Supabase через `create_public_lead` RPC

**Что делать:**
- [ ] Создать JavaScript для обработки формы
- [ ] Валидация полей:
  - Имя не пусто
  - Email валиден
  - Телефон не пусто

- [ ] При отправке:
  - Показать "Отправка..."
  - Вызвать `create_public_lead` RPC
  - Показать "Спасибо" сообщение
  - Очистить форму

- [ ] Обработка ошибок:
  - Показать ошибку если не удалось отправить
  - Кнопка повтора

**Код структура:**
```javascript
// landing.js
async function submitLeadForm() {
  const formData = { /* собрать из инпутов */ };
  
  try {
    const { error } = await sb.rpc('create_public_lead', {
      p_customer_name: formData.name,
      p_customer_phone: formData.phone,
      p_contact_email: formData.email,
      p_company_name: formData.company,
      p_message: formData.message
    });
    
    if (error) throw error;
    showSuccess('Спасибо! Мы свяжемся с вами');
  } catch (err) {
    showError('Ошибка при отправке');
  }
}
```
